# Hotel Room Booking Chatbot (SaraBot)

SaraBot is a Python-based, rule-driven chatbot that simulates a hotel room booking assistant
through a text-based conversational interface.

This project focuses on practical Natural Language Processing (NLP) concepts and
state-based dialogue design rather than machine learning models.

---

## Features
- Step-by-step conversational booking flow
- Room selection by number or partial room name
- Rule-based intent handling
- Input validation and user-friendly prompts
- Modular and extensible design

---

##  Tech Stack
- Python 3
- Basic NLP (string matching & rules)
- Command-line interface (CLI)
